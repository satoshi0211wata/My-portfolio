Chat-Space

アプリの説明

はじめて世界に公開したアプリケーション。Chat-Spaceは複数人によるチャット機能を搭載したwebアプリケーション。
TEST用Log in：Eメール/login@mail パスワード/aaaaaaaa
アプリを見る　>>

扱ったツール

Mac /Ruby on Rails /Ruby /HTML(haml) /CSS(sass) /JavaScript /jquery / SQL/ git/ github
/Linux/ AWS(EC2,IAM,S3)/ Node.js/ MySQL/ Unicorn/ Nginx/ Capistrano

工夫した点

◯チャットメッセージの送信機能（非同期処理）にAjax通信,jbuilderを使用して最小限のデータアクセスで投稿、反映できるユーザーアクセシビリティにした。
◯チャットグループ作成機能の中で非同期通信でメンバー検索、追加したメンバーの反映（インクルメンタルサーチ、追加）を行えるようにした。
◯AWSを使ってデプロイ。EC2インスタンス内で、OSはLinux、webサーバーはNginx、アプリケーションサーバーはUnicorn、データベースはMySQL、Capistranoを使いGitHub経由で自動デプロイをできるように環境構築した。画像はS3に保存。
